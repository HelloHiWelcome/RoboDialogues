# RoboDialogues
Robos in Movies

## Description
This project extracts dialogues for AI/robot characters from movies, stores them in a database, and offers easy access to these dialogues for analysis.

## Sources
Cornell Movie-Dialogs Corpus: https://convokit.cornell.edu/documentation/movie.html

Movie Scripts Corpus: https://www.kaggle.com/datasets/gufukuro/movie-scripts-corpus

MovieSum Dataset: https://huggingface.co/datasets/rohitsaxena/MovieSum
